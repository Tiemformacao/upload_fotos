Projeto de upload de imagens

Projeto desenvolvido objetivando o aprendizado em relação a funcionalidade de upload de imagens.

O usuário seleciona a imagem.
Envia.
É direcionado à uma página para escolher entre realizar o upload de outra imagem ou buscar no banco de dados a imagem enviada.
Caso escolha buscar imagens enviadas, terá um campo onde poderá digitar o nome da foto.
Será mostrado todas as imagens que contenha o nome ou parte do nome digitado.
Nesta última página, contém um link com 'voltar', que será direcionado para página inicial de upload de fotos.
